Welcome to Epycom's documentation!
==================================

Epycom (c) is a python package developed by `Biomedical Engineering Department <http://bme.fnusa.cz>`_ of International Clinical Research Center within St. Anne's University Hospital in Brno. The main focus of this package is to provide tools for automated feature extraction from EEG and iEEG. The package also provides tools for automated noise detection, validation of the algorithms and machine learning tools for localization of epileptogenic foci.
